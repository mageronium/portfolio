HTML (HyperText Markup Language) Website project

(Project started November 5 2018 completed November 30 2018, CSP (Computer Science Principles))

The HTML Website project, which this very website is the product of, is a project designed to test student's abilities in using HTML and presenting themselves professionally on a platform. The constraints of this project were primarily time based and challenges were not typically caused by inexperience as students were given the documentation to the bootstrap CSS (Cascading Style Sheets). The deliverables for this project included fully annotated code and this website. As this was an individual project I performed all of the research, design, annotation, and coding required to build the website and complete the project. However as a relative novice, I was quite dependent on the bootstrap documentation, but as I learnt more about HTML through this project I was able to code more effectively and independent of the documentation. After the website was built, I added on several animations across the site which required intense research through the bootstrap documentation and hours of trial and error, but were ultimately worthwhile in the end. This website overall is the composition of my ability to persevere and learn via hands-on-experience by analysing pre existing code.
